# Hermes Agent — A Field Guide to Eight Powers

An interactive single-page field guide introducing eight core Hermes Agent powers.

## Preview locally

Open `index.html` in a browser, or run a simple local server:

```bash
python3 -m http.server 8000
```

Then visit <http://localhost:8000>.

## GitHub Pages

This repository is ready to publish with GitHub Pages from the `main` branch root.
